# ListingSitesTeams
ListingSitesTeams is an application integrated into Microsoft Teams for managing clients and teams. It allows managing teams and associated members. This project is designed to facilitate the management of teams and clients directly from the Microsoft Teams interface, providing a centralized and efficient solution.

## Main Features

- **Integrated application in Microsoft Teams**: Allows users to manage clients and teams without leaving the Teams environment.
- **Team and member management**: Add, modify, and delete teams and members, and assign specific roles and responsibilities.
- **Real-time updates**: Changes made are immediately visible to all team members.
- **Responsive user interface**: Designed to be used on various devices, including desktops, tablets, and smartphones.
- **User authentication and authorization**: Secure access to the application with robust authentication and authorization mechanisms.

## Technology Stack

- **JavaScript**: Main programming language used for application development.
- **CSS**: Used for styling and layout of the application.
- **React Native**: Framework used to build the user interface.
- **NodeJS API**: Used to create the backend of the application, handle requests, and manage data.

## Installation

To install and run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/anais-mez/ListingSitesTeams.git
    cd ListingSitesTeams
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm start
    ```

## Usage

After starting the development server, open your browser and go to `http://localhost:3000` to see the application in action. You can then start adding teams, managing members, and seeing real-time updates.
